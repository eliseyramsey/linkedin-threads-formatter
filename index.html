How to Use

Formatting

Bold and Italic work only with Latin characters (a-z, A-Z). Strikethrough and underline work with any characters, including Cyrillic. Undo (Ctrl+Z) and redo (Ctrl+Shift+Z) available via buttons or keyboard.

Auto-Split

For Threads, text is automatically split into parts of 500 characters with numbering 1/n, 2/n, etc.

///

Manual Split

Insert /// in text to specify thread break point. Takes priority over auto-split.

Copying

LinkedIn — one button copies the entire post. Threads — separate button for each thread segment.
