## 3. Ручное разбиение на треды
**Символ разрыва:** `///` (три слеша)

**Логика:**
- При встрече `///` в тексте — принудительно начинать новый блок
- Символ `///` не включается в финальный текст
- Работает вместе с автоматическим разбиением по 500 символов
- Приоритет: ручной разрыв > автоматический

**Пример:**
```
Первый пост о важной теме.
///
Второй пост с продолжением.
```
Результат: 2 отдельных блока, независимо от длины.

## 7 How To (внизу страницы)

**Обязательные разделы:**

How to use Post Formatter:

1. **Unicode и кириллица**
   - Объяснение: Unicode bold/italic работает только с латиницей
   - Почему: Unicode не имеет математических символов для кириллицы
   - Рекомендация: Использовать английский текст для форматирования

2. **Работа с Threads**
   - Автоматическая разбивка по 500 символов
   - Сохранение целостности предложений и слов
   - Нумерация блоков (1/n, 2/n, ...)

   - Как использовать символ разрыва
   - Примеры использования
   - Приоритет над автоматической разбивкой

4. **Копирование контента**
